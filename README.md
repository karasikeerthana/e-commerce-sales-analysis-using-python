# E-Commerce-Sales-Analysis-using-Python

## 📌 Overview
This project analyzes e-commerce sales data using Python to uncover key business insights and trends.

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Key Features
- Data cleaning and preprocessing
- Feature engineering (Year, Month, Profit Margin)
- KPI calculation (Sales, Profit, Orders)
- Sales trend analysis
- Regional and category performance analysis
- Identification of loss-making products

## 📈 Key Insights
- West region contributes highest sales
- Some products generate losses despite high sales
- Sales trends vary across months
- Profit is not always proportional to sales

## 📂 Project File
- analysis.ipynb (Jupyter Notebook)
